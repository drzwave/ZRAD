# Control tinySA using a python script

tinySA can be controlled via the USB serial interface from a PC. Here are some examples.

## Preparation

    $ cd python
    $ pip3 install -r requirements.txt

## Run

### Plot default scan.

    $ ./tinySA.py -p

### Capture display

    $ ./tinySA.py -C out.png

### Show usage.

    $ ./tinySA.py -h
